# PACMAN

A game designed by c and OpenGL

## Features

- the baisc pacman game with some modifications

## Run Locally

Clone the project

```bash
  git clone https://github.com/PrBr20/MediAid.git
```

Go to the project directory and run the PACMAN.cpp file

```bash
  g++ -o PACMAN.cpp && ./a.out
```
